# Library Management System (C++ Console Application)

A console-based Library Management System developed in C++ as a Semester 1 academic project.

This system provides an efficient solution for managing books, members, and library transactions through an interactive menu-driven interface.

---

# Project Overview

The system helps librarians perform essential library operations such as:

- Book management
- Member registration
- Book issuing and returning
- Searching records
- Updating information
- Reporting issued books

---

# Key Features

## User-Friendly Menu System

- Numbered menu navigation
- Input validation
- Error prevention
- Sound feedback using Beep()

---

## Book Management

- Add books
- Update books
- Remove books
- View all books
- Search by title
- Search by author
- Search by category

---

## Member Management

- Register new members
- Update member details
- Prevent duplicate IDs
- Student / Teacher distinction

---

## Transaction System

- Issue books
- Return books
- Track issued books
- Issue limits:

**Students:** 2 Books  
**Teachers:** 3 Books

---

## Reporting System

- View issued books count
- Detailed status information
- Search reports

---
